# Lisp interpreter

## Features

**Implemented:**

- [x] Supports function calls

```lisp
(print 10)
(print (add 5 10))
```

**Todo:**

- [ ] Add support for variables
- [ ] Add support for functions

## Development

### Prerequisites

- Node.js and npm (or yarn)

### Setup

1. Clone the repo

```bash
git clone https://github.com/ivteplo/lisp-interpreter
```

2. Install dependencies

```bash
npm install
# or, if you prefer yarn:
yarn install
```
